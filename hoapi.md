## 版本类型说明

| 类型    | 说明                                 | 作用                           |
| ------- | ------------------------------------ | ------------------------------ |
| Canary  | 早期体验版本，特性功能待稳定         | 可反馈问题，华为全力解决       |
| Beta    | 公开发布的Beta版本，特性功能待稳定   | 可反馈使用体验，改进产品       |
| Release | 正式发布版本，承诺质量，特性功能稳定 | 可基于新功能完成开发，发布应用 |
